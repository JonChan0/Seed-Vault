# Obsidian Setup

Seed Vault is pre-configured for Obsidian with graph view colors, templates, and backlink settings ready out of the box.

## Opening the Vault

Obsidian → File → Open Vault → select the vault folder.

## Graph View Colors

The `type` frontmatter field controls node color in Obsidian's graph view:

| Type | Color | Used For |
|------|-------|----------|
| `concept` | Blue | Synthesized concept articles |
| `source-summary` | Green | One summary per `raw/` file |
| `topic` | Purple | Cluster hub pages |
| `visualization` | Orange | HTML viz wrapper pages |

## Templates

Obsidian templates are stored in `_templates/`. Available templates:

- `concept.md` — concept article scaffold
- `source-summary.md` — source summary scaffold
- `visualization.md` — viz wrapper scaffold

These are for reference — the LLM uses them when writing new articles. You can also use them manually via Obsidian's template inserter (Ctrl/Cmd+T).

## Enabled Features

The `.obsidian/` config has these enabled by default:

- **Backlinks** — see all articles that link to the current note
- **Properties** — YAML frontmatter displayed as structured fields
- **Graph view** — visual map of article interconnections

## Viewing HTML Visualizations

HTML visualizations in `viz/` are embedded in wrapper `.md` pages via `<iframe>`. To view them in Obsidian:

1. Install the **HTML Reader** community plugin (recommended), OR
2. Open the `.html` file from Obsidian's file explorer — opens in your browser

Mermaid diagrams in regular `.md` files render natively — no plugin needed.

## Obsidian Bases (`_index.base`)

`wiki/_index.base` is an Obsidian Bases view of the index auto-populated from frontmatter. It provides a filterable, sortable table view of all articles by type, status, and tags. Do not edit this file manually.

## Recommended Plugins

| Plugin | Purpose | Required? |
|--------|---------|-----------|
| HTML Reader | Render `<iframe>` viz pages in Obsidian | Optional |
| Obsidian Web Clipper | Clip web pages directly into `raw/` as markdown | Optional |
| Dataview | Advanced queries over frontmatter | Optional |

**Obsidian Web Clipper tip:** Tags clipped pages with `#Clippings` automatically — pairs naturally with `vault-ingest`.
